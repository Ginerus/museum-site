О проекте

```

Проект разработан в рамках дипломной работы по специальности «Информационные системы и программирование».

Приложение построено по клиент-серверной архитектуре: WPF-клиент взаимодействует с ASP.NET Web API, а данные хранятся в SQL Server.

```

Возможности

```
  • Просмотр экспонатов
  • Поиск по экспонатам
  • Просмотр подробной информации
  • Авторизация администратора
  • Добавление и редактирование данных
  • Работа через Web API
```
